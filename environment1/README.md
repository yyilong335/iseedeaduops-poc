# Environment

run
```
sudo sh collect.sh
```
CPU information will be output to `cpuid.txt`.

System information will be output to `lsb_release_a.txt`.

Architecture information will be output to `lscpu.txt`.

Kernel information will be output to `uname_r.txt`.
